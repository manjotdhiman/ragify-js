# Configuration

This page details all configuration options available in Ragify.js.

## Basic Configuration

The basic configuration includes the essential settings needed to get started:

```typescript
const config = {
  llmProvider: "openai",
  embeddingProvider: "openai",
  vectorStore: "qdrant",
  apiKeys: {
    openai: "your-openai-key",
    qdrant: "your-qdrant-key"
  }
};
```

## Advanced Configuration

### Embedding Configuration

```typescript
const config = {
  // ... basic config
  embeddingModel: "text-embedding-ada-002", // OpenAI model
  embeddingProvider: "openai" | "cohere" | "huggingface",
  apiKeys: {
    openai: "your-openai-key",
    cohere: "your-cohere-key", // Optional
    huggingface: "your-huggingface-key" // Optional
  }
};
```

### Chunking Configuration

Control how documents are split into chunks:

```typescript
const config = {
  // ... basic config
  chunkingConfig: {
    maxTokens: 200, // Maximum tokens per chunk
    overlap: 50, // Number of tokens to overlap between chunks
    preserveSentences: true // Whether to keep sentences intact
  }
};
```

### Search Configuration

Configure how results are retrieved and filtered:

```typescript
const config = {
  // ... basic config
  searchConfig: {
    topK: 5, // Number of results to return
    minScore: 0.7 // Minimum similarity score (0-1)
  }
};
```

### Retry Configuration

Configure retry behavior for API calls:

```typescript
const config = {
  // ... basic config
  retryConfig: {
    maxRetries: 3, // Maximum number of retries
    initialDelay: 1000, // Initial delay in milliseconds
    maxDelay: 10000 // Maximum delay in milliseconds
  }
};
```

### Vector Store Configuration

Configure the vector store settings:

```typescript
const config = {
  // ... basic config
  vectorStoreConfig: {
    baseURL: "http://localhost:6333", // Qdrant server URL
    collection: "ragify", // Collection name
    cacheConfig: {
      maxSize: 1000, // Maximum number of items in cache
      ttl: 5 * 60 * 1000 // Cache TTL in milliseconds (5 minutes)
    }
  }
};
```

### Concurrency Configuration

Control how many chunks are processed concurrently:

```typescript
const config = {
  // ... basic config
  maxConcurrentChunks: 10 // Maximum number of concurrent chunk operations
};
```

const config = {

## Environment Variables

For sensitive configuration like API keys, you can use environment variables:

```typescript
const config = {
  llmProvider: "openai",
  embeddingProvider: "openai",
  vectorStore: "qdrant",
  apiKeys: {
    openai: process.env.OPENAI_API_KEY,
    qdrant: process.env.QDRANT_API_KEY
  }
};
```

## Type Safety

All configuration options are fully typed. You can import the types from Ragify.js:

```typescript
import { RAGEngineConfig } from "ragify-js";

const config: RAGEngineConfig = {
  // ... your configuration
};
```
