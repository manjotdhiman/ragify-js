## Quick Start

```bash
yarn add ragify-js
```

```typescript
import { createRAGEngine } from "ragify-js";

const rag = createRAGEngine({
  llmProvider: "openai",
  embeddingProvider: "openai",
  vectorStore: "qdrant",
  apiKeys: {
    openai: "your-openai-key",
    qdrant: "your-qdrant-key",
  },
});

// Ingest documents
await rag.ingestDocuments([
  {
    id: "doc1",
    text: "This is a test document",
    metadata: { source: "example" },
  },
]);

// Query the documents
const results = await rag.query("What is this document about?", 5, 0.7);
console.log(results);
```
